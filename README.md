# ❤️ Два года вместе

> Некоторые сайты создаются для работы.  
> Этот — для одного человека.

Этот проект — небольшой подарок, который я сделал в честь нашей второй годовщины.

Здесь нет сложных алгоритмов или революционных технологий. Здесь есть кое-что важнее: воспоминания, моменты, фотографии, музыка и чувства, которые хотелось сохранить немного необычным способом.

Если вы случайно нашли этот репозиторий — добро пожаловать. Это не коммерческий проект и не работа для портфолио. Это просто сайт, созданный с душой для человека, который стал очень важной частью моей жизни.

## ✨ Что внутри

- ⏳ Счётчик времени, проведённого вместе
- 📖 История наших отношений
- 🖼️ Галерея фотографий
- ❤️ Тёплые признания и нежные слова
- 🎵 Музыка, связанная с нашими воспоминаниями
- 🎉 Анимации, детали и маленькие пасхалки

## 🛠️ Использованные технологии

- HTML5
- CSS3
- JavaScript (Vanilla)

Без фреймворков. Просто код, немного фантазии и очень много желания сделать приятный подарок.

## 💌 Личное

Соня.

Если ты читаешь это, значит подарок уже нашёл своего адресата.
